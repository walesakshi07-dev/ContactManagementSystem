A web-based Contact Management System developed using HTML, CSS, PHP and MySQL to manage user contact information digitally.
# 📞 Contact Management System

## 📌 Project Overview

Contact Management System is a web-based application developed to store, manage, and organize contact information digitally. This system allows users to add, update, delete, and search contact details easily.

The main purpose of this project is to replace manual contact management and provide a simple, efficient, and secure way to manage personal and professional contacts.

---

## 🎯 Project Aim

The aim of this project is to develop a digital contact management platform that helps users maintain contact details in an organized manner.

---

## 🎯 Objectives

- To store contact information digitally.
- To provide easy access to saved contacts.
- To add, update, delete, and search contacts.
- To reduce manual record keeping.
- To maintain contact data securely.

## 🗄 Database Tables

### Users Table

Stores user login information.

Fields:
- User_ID
- Name
- Email
- Password


### Contacts Table

Stores contact details.

Fields:
- Contact_ID
- User_ID
- Contact_Name
- Mobile_Number
- Email
- Address
- Created_Date

---

## 🔄 Working Process

1. User registers into the system.
2. User logs in using email and password.
3. User adds contact details.
4. Contact information is stored in MySQL database.
5. User can search, update, or delete contacts whenever required.
